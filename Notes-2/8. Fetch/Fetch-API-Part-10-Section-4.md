## Interview Questions

**Q:** What are the advantages of an API service layer?

**A:** It improves maintainability, consistency, testability, reuse, and
separation of concerns.

**Q:** Why centralize error handling?

**A:** It ensures consistent behavior and reduces duplicated logic
across the application.

## Debugging Checklist

-   Verify the base URL.
-   Confirm request headers.
-   Test CRUD helpers individually.
-   Check authentication and token refresh.
-   Inspect requests in the browser Network tab.
