
# Lesson 6: Output

## Phase 1: Programming Fundamentals

## Learning Objectives

By the end of this lesson, you will be able to:

- Explain what output is.
- Display information using `console.log()`.
- Use `alert()` to show messages.
- Write content to a web page with `document.body.textContent`.
- Combine text with variables.
- Choose the appropriate output method.

---

# What Is Output?

**Output** is any information a program presents after processing data.

Examples:

- Text on a screen
- A notification
- A printed receipt
- A generated report

Mental model:

```text
Input → Processing → Output
```

---

# console.log()

The most common output while learning JavaScript.

```javascript
console.log("Hello, JavaScript!");
```

You can also print variables.

```javascript
const name = "Alice";
console.log(name);
```

Or multiple values.

```javascript
const age = 25;
console.log("Age:", age);
```

---

# alert()

Displays a popup dialog.

```javascript
alert("Welcome!");
```

Use it sparingly because it interrupts the user.

---

# Writing to the Web Page

You can display text directly on the page.

```javascript
document.body.textContent = "Hello from JavaScript!";
```

Later you'll learn richer DOM manipulation.

---

# Combining Output

```javascript
const city = "Nairobi";
console.log("City:", city);

document.body.textContent = "Welcome to " + city;
```

---

# Choosing the Right Output

| Method | Best Use |
|---|---|
| `console.log()` | Debugging and learning |
| `alert()` | Simple notifications |
| `document.body.textContent` | Showing content on a web page |

---

# Common Beginner Mistakes

- Expecting `console.log()` to appear on the page.
- Forgetting to open Developer Tools.
- Overusing `alert()`.
- Misspelling `console.log`.

---

# Best Practices

- Use `console.log()` while developing.
- Display information on the page for users.
- Keep messages clear and descriptive.

---

```

```javascript

```

```javascript

```

3. Display "Learning JavaScript" on the page.

5. Print a label and value together using `console.log()`.

---

Requirements:

- Store a user's name in a variable.
- Print a welcome message to the console.
- Show a popup greeting.
- Display the same greeting on the web page.

---

# Summary

You learned:

- Output presents information to users or developers.
- `console.log()` is ideal for debugging.
- `alert()` creates popup messages.
- `document.body.textContent` displays text on a page.
- Different output methods have different purposes.
