# 1. The Problem

So far:

```text
You add items → refresh page → everything disappears ❌
```

Why?

Because:

```text
State exists only in memory (RAM)
```

When the page reloads:

```text
JavaScript restarts → state resets
```

---

# 2. What is localStorage?

`localStorage` is a browser feature that allows you to:

* Store data in the browser
* Keep data even after refresh
* Keep data even after closing the browser

---

## Key Properties

| Feature      | Description                      |
| ------------ | -------------------------------- |
| Storage type | Key-value                        |
| Data type    | Strings only                     |
| Persistence  | Permanent until manually cleared |
| Size limit   | ~5MB                             |

---

# 3. Important Rule

> ⚠️ localStorage only stores **strings**

This means:

```js
localStorage.setItem("products", products);
```

❌ WRONG

# 5. Saving State to localStorage

## What You Will Learn

* How to store your products array
* When to save state
* Why saving must happen after every change

---

## Concept Explanation

Whenever your state changes:

```text
Create → Update → Delete
```

You must save:

```text
state → localStorage
```

---

## Syntax Breakdown

```js
localStorage.setItem("products", JSON.stringify(products));
```

* `"products"` → key name
* `JSON.stringify(products)` → converts array to string

---

## Example 1: Basic

```js
const numbers = [1, 2, 3];

localStorage.setItem(
  "numbers",
  JSON.stringify(numbers)
);
```

---

## Example 2: Array of Objects

```js
const products = [
  { id: 1, name: "Laptop" },
];

localStorage.setItem(
  "products",
  JSON.stringify(products)
);
```

---

## Example 3: With CRUD

```js
function saveProducts() {
  localStorage.setItem(
    "products",
    JSON.stringify(products)
  );
}
```

Call it after every mutation:

```js
createProduct();
saveProducts();

updateProduct();
saveProducts();

deleteProduct();
saveProducts();
```

# 6. Loading State from localStorage

## What You Will Learn

* How to load saved data when the page starts
* How to avoid crashes if no data exists

---

## Concept Explanation

When your app loads:

```text
Check localStorage → load saved data → use as state
```

---

## Syntax Breakdown

```js
const stored = localStorage.getItem("products");
```

* Returns a string OR `null`

---

## Example 1: Basic

```js
const stored =
  localStorage.getItem("numbers");

const numbers =
  JSON.parse(stored);

console.log(numbers);
```

---

## Example 2: Safe Loading

```js
const stored =
  localStorage.getItem("products");

let products = [];

if (stored !== null) {
  products = JSON.parse(stored);
}
```

---

## Example 3: Best Practice

```js
function loadProducts() {
  const stored =
    localStorage.getItem("products");

  if (stored === null) {
    return [];
  }

  try {
    return JSON.parse(stored);
  } catch (error) {
    console.error(
      "Invalid stored data"
    );
    return [];
  }
}
```

# 7. Integrating with Your CRUD App

## Step 1: Initialize State

Replace:

```js
let products = [];
```

With:

```js

```

## Step 2: Save After Every Change

Update your CRUD functions:

### Create

```js
function createProduct(data) {
  const newProduct = { id: nextId, ...data };

  products = [...products, newProduct];

  saveProducts();
}
```

---

### Update

```js
products = products.map(...);

saveProducts();
```

---

### Delete

```js
products = products.filter(...);

saveProducts();
```

# 8. Full Integration Example

```js
function saveProducts() {
  localStorage.setItem(
    "products",
    JSON.stringify(products)
  );
}

function loadProducts() {
  const stored =
    localStorage.getItem("products");

  if (!stored) return [];

  return JSON.parse(stored);
}

let products = loadProducts();

function createProduct(data) {
  products = [...products, data];

  saveProducts();
  renderProducts();
}
```

# 9. Handling IDs with Stored Data

## Problem

After reload:

```js
nextProductId resets ❌
```

---

## Solution

Calculate next ID dynamically:

```js
function getNextId() {
  if (products.length === 0) {
    return 1;
  }

  return Math.max(
    ...products.map(p => p.id)
  ) + 1;
}
```
