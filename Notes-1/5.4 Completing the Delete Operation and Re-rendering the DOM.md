# 20. Completing the Delete Operation

## 20.1 What You Will Learn

Delete is complete only when all related state and DOM changes are handled correctly.

Removing an object from an array is only one part.

A complete Delete operation should:

1. Identify the correct object.
2. Protect against invalid or stale IDs.
3. Ask for user confirmation.
4. Remove the object from JavaScript state.
5. Repair related UI state.
6. Re-render the page.
7. Update derived information.
8. Show clear feedback.

---

# 20.2 Complete Delete Architecture

A well-structured Delete workflow may use these functions:

```js
findProductById(productId);
findProductIndexById(productId);
confirmProductDeletion(product);
deleteProductById(productId);
handleDeleteRequest(productId);
resetFormToCreateMode();
renderProducts();
showError(message);
showSuccess(message);
showInfo(message);
```

Each function should have one clear responsibility.

# 20.4 Rendering Delete Buttons

Each product card needs a Delete button.

```js
const deleteButton =
  document.createElement("button");

deleteButton.type = "button";
deleteButton.textContent = "Delete";

deleteButton.dataset.action =
  "delete";

deleteButton.dataset.id =
  product.id;
```

Generated HTML:

```html
<button
  type="button"
  data-action="delete"
  data-id="205"
>
  Delete
</button>
```

The button stores the stable product ID, not the array index.

---

# 20.5 Why Buttons Should Store IDs

Suppose Phone is currently at index `1`.

After Laptop is deleted, Phone moves to index `0`.

Its ID remains:

```text
205
```

If a button stored index `1`, that value could become stale.

Stable IDs allow the application to calculate the current index when needed.

# 20.6 Event Delegation

Attach one listener to the parent list:

```js
productList.addEventListener(
  "click",
  handleProductListClick
);
```

Handler:

```js
function handleProductListClick(
  event
) {
  const clickedButton =
    event.target.closest(
      "button"
    );

  if (clickedButton === null) {
    return;
  }

  const action =
    clickedButton.dataset.action;

  if (action !== "delete") {
    return;
  }

  const productId =
    Number(
      clickedButton.dataset.id
    );

  handleDeleteRequest(
    productId
  );
}
```

```js

    product
```

* Display an error.

# 20.11 Low-Level State Deletion

```js
function deleteProductById(
  productId
) {
  const productIndex =
    findProductIndexById(
      productId
    );

  if (productIndex === -1) {
    return {
      success: false,
      deletedProduct: null,
      message:
        "The selected product no longer exists.",
    };
  }

  const removedProducts =
    products.splice(
      productIndex,
      1
    );

  return {
    success: true,
    deletedProduct:
      removedProducts[0],
    message:
      "Product deleted successfully.",
  };
}
```

This function:

* Finds the current index.
* Checks for `-1`.
* Removes one object.
* Returns a descriptive result.

---

# 20.12 Why Re-find the Index?

Even if the product was found before confirmation, calculate the index again inside the state-deletion function.

```js
const productIndex =
  findProductIndexById(
    productId
  );
```

This keeps the mutation close to the lookup and avoids using a long-lived index.

After successful deletion:

```js
if (
```

This prevents the form from continuing to edit a missing product.

# 20.14 Preserve an Unrelated Edit

Suppose:

```js
editingProductId = 101;
```

The user deletes:

```js
productId = 205;
```

The condition is false:

```js
editingProductId ===
  deletedProduct.id
```

The Laptop edit should remain active.

Do not reset all form state after every deletion.

# 20.15 Re-render After Successful Deletion


# 20.16 Why Rendering Must Happen After `splice()`

Incorrect:

```js
renderProducts();

products.splice(
  productIndex,
  1
);
```

The deleted product remains visible.

Correct:

```js
products.splice(
  productIndex,
  1
);

renderProducts();
```

The render function reads the updated state.

# 20.18 Recalculating Inventory Value with a Plain Loop

A product's inventory value is:

```text
price × quantity
```

Complete inventory total:

```js
function calculateInventoryValue() {
  let totalValue = 0;

  for (
    let index = 0;
    index < products.length;
    index++
  ) {
    totalValue +=
      products[index].price *
      products[index].quantity;
  }

  return totalValue;
}
```

Render it:

```js
inventoryValue.textContent =
  `Inventory value: KSh ${calculateInventoryValue().toLocaleString()}`;
```

Deletion changes this derived total.

# 20.19 Counting Out-of-Stock Products

```js
function countOutOfStockProducts() {
```

# 20.22 Complete Delete Request Function

```js
function handleDeleteRequest(
  productId
) {
  clearMessages();

  if (
    !isValidProductId(
      productId
    )
  ) {
    showError(
      "The selected product ID is invalid."
    );

    return false;
  }

  const product =
    findProductById(
      productId
    );

  if (product === null) {
    showError(
      "The selected product no longer exists."
    );

    renderProducts();

    return false;
  }

  const shouldDelete =
    confirmProductDeletion(
      product
    );

  if (!shouldDelete) {
    showInfo(
      `${product.name} was not deleted.`
    );

    return false;
  }

  const result =
    deleteProductById(
      productId
    );

  if (!result.success) {
    showError(
      result.message
    );

    renderProducts();

    return false;
  }

  if (
    editingProductId ===
    result.deletedProduct.id
  ) {
    resetFormToCreateMode();
  }

  renderProducts();

  showSuccess(
    `${result.deletedProduct.name} was deleted successfully.`
  );

  return true;
}
```

---

# 20.23 Why Return a Boolean?

The request function returns:

```js
true
```

when an item was deleted.

It returns:

```js
false
```

when:

* The ID was invalid.
* The product was missing.
* The user canceled.
* Deletion failed.

This makes the function easier to test and coordinate.

# 20.28 Complete Delete Flow Diagram

```text
Delete button clicked
        ↓
Read dataset ID
        ↓
Convert ID to number
        ↓
Validate positive integer
        ↓
Find product object
        ↓
Product missing?
  Yes → show error and stop

        ↓
```

# 20.29 Three Complete Examples

## Example 1: Complete Delete Without the DOM

### Problem

Delete a product by ID and return a structured result.

### Input Data

```js
let products = [
  {
    id: 1,
    name: "Laptop",
  },
  {
    id: 2,
    name: "Phone",
  },
  {
    id: 3,
    name: "Keyboard",
  },
];
```

### Complete Code

```js
let products = [
  {
    id: 1,
    name: "Laptop",
  },
  {
    id: 2,
    name: "Phone",
  },
  {
    id: 3,
    name: "Keyboard",
  },
];

function findProductIndexById(
  productId
) {
  for (
    let index = 0;
    index < products.length;
    index++
  ) {
    if (
      products[index].id ===
      productId
    ) {
      return index;
    }
  }

  return -1;
}

function deleteProductById(
  productId
) {
  const index =
    findProductIndexById(
      productId
    );

  if (index === -1) {
    return {
      success: false,
      deletedProduct: null,
      message:
        "Product not found.",
    };
  }

  const removed =
    products.splice(
      index,
      1
    );

  return {
    success: true,
    deletedProduct:
      removed[0],
    message:
      "Product deleted.",
  };
}

const result =
  deleteProductById(2);

console.log(result);
console.log(products);
```

### Expected Console Output

```text
{
  success: true,
  deletedProduct: {
    id: 2,
    name: "Phone"
  },
  message: "Product deleted."
}

[
  {
    id: 1,
    name: "Laptop"
  },
  {
    id: 3,
    name: "Keyboard"
  }
]
```

### Final State

```js
[
  {
    id: 1,
    name: "Laptop",
  },
  {
    id: 3,
    name: "Keyboard",
  },
]
```

### Common Mistakes

* Returning the whole `removed` array instead of one object.
* Failing to check for `-1`.
* Using the ID as the `splice()` index.

### Small Modification Challenge

Try deleting ID `999`.

Confirm that state does not change.

Delete a product and recalculate the product count and total inventory value.

```js
  {
```

```html
      editingProductId = null;
    }

    function renderSummary() {
      productCount.textContent =
        `Total products: ${products.length}`;

      inventoryValue.textContent =
        `Inventory value: KSh ${calculateInventoryValue().toLocaleString()}`;

      outOfStockCount.textContent =
        `Out of stock: ${countOutOfStockProducts()}`;
    }

    function renderEmptyState() {
      const emptyState =
        document.createElement("li");

      emptyState.classList.add(
        "empty-state"
      );

      emptyState.textContent =
        "No products available.";

      productList.appendChild(
        emptyState
      );
    }

    function renderProductCard(
      product,
      index
    ) {
      const card =
        document.createElement("li");

      const name =
        document.createElement("h2");

      const details =
        document.createElement("p");

      const status =
        document.createElement("p");

      const actions =
        document.createElement("div");

      const editButton =
        document.createElement("button");

      const deleteButton =
        document.createElement("button");

      card.classList.add(
        "product-card"
      );

      actions.classList.add(
        "product-actions"
      );

      card.dataset.id =
        product.id;

      name.textContent =
        `${index + 1}. ${product.name}`;

      details.textContent =
        `KSh ${product.price.toLocaleString()} — ` +
        `Quantity: ${product.quantity} — ` +
        `Category: ${product.category}`;

      if (product.quantity === 0) {
        status.textContent =
          "Out of stock";
      } else {
        status.textContent =
          "In stock";
      }

      editButton.type =
        "button";

      editButton.textContent =
        "Edit";

      editButton.dataset.action =
        "edit";

      editButton.dataset.id =
        product.id;

      deleteButton.type =
        "button";

      deleteButton.textContent =
        "Delete";

      deleteButton.dataset.action =
        "delete";

      deleteButton.dataset.id =
        product.id;

      actions.append(
        editButton,
        deleteButton
      );

      card.append(
        name,
        details,
        status,
        actions
      );

      productList.appendChild(
        card
      );
    }

    function renderProducts() {
      productList.innerHTML = "";

      renderSummary();

      if (products.length === 0) {
        renderEmptyState();
        return;
      }

      for (
        let index = 0;
        index < products.length;
        index++
      ) {
        renderProductCard(
          products[index],
          index
        );
      }
    }

    function handleDeleteRequest(
      productId
    ) {
      clearMessages();

      if (
        !isValidProductId(
          productId
        )
      ) {
        showError(
          "The selected product ID is invalid."
        );

        return false;
      }

      const product =
        findProductById(
          productId
        );

      if (product === null) {
        showError(
          "The selected product no longer exists."
        );

        renderProducts();

        return false;
      }

      const shouldDelete =
        confirmProductDeletion(
          product
        );

      if (!shouldDelete) {
        showInfo(
          `${product.name} was not deleted.`
        );

        return false;
      }

      const result =
        deleteProductById(
          productId
        );

      if (!result.success) {
        showError(
          result.message
        );

        renderProducts();

        return false;
      }

      if (
        editingProductId ===
        result.deletedProduct.id
      ) {
        resetFormToCreateMode();
      }

      renderProducts();

      showSuccess(
        `${result.deletedProduct.name} was deleted successfully.`
      );

      return true;
    }

    function handleProductListClick(
      event
    ) {
      const clickedButton =
        event.target.closest(
          "button"
        );

      if (
        clickedButton === null
      ) {
        return;
      }

      const action =
        clickedButton.dataset.action;

      const productId =
        Number(
          clickedButton.dataset.id
        );

      if (action === "edit") {
        if (
          !isValidProductId(
            productId
          )
        ) {
          showError(
            "Invalid product ID."
          );

          return;
        }

        editingProductId =
          productId;

        clearMessages();

        showInfo(
          `Editing product ID ${productId}.`
        );

        return;
      }

      if (action === "delete") {
        handleDeleteRequest(
          productId
        );
      }
    }

    productList.addEventListener(
      "click",
      handleProductListClick
    );

    renderProducts();
  </script>
</body>
</html>
```

## Key Takeaways

* A complete Delete operation includes selection, validation, confirmation, mutation, UI-state repair, rendering, and feedback.
* Delete buttons should store stable IDs instead of array indexes.
* DOM IDs must be converted and validated.
* The selected product should exist before confirmation begins.
* Cancellation must stop the operation without changing state.
* The current array index should be calculated close to `splice()`.
* Always check for `-1`.
* `splice(index, 1)` removes exactly one item and mutates the array.
* The deleted object can be captured for feedback.
* Edit mode should reset only when its product is deleted.
* Re-rendering keeps the DOM synchronized with state.
* Counts, totals, stock information, and empty-state output should be derived from the current array.
