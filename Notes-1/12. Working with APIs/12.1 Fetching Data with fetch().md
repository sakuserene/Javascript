# 3. GET Request (Read Data)

---

## What You Will Learn

How to fetch data from server.

---

## Example 1 (Basic)

```js
fetch("https://jsonplaceholder.typicode.com/posts")
  .then(response => response.json())
  .then(data => console.log(data));
```

---

## Flow

```text
Request → Response → Convert to JSON → Use data
```

---

## Example 2 (Async/Await)

```js
async function loadData() {
  const response = await fetch("https://jsonplaceholder.typicode.com/posts");
  const data = await response.json();
  console.log(data);
}
```

---

## Example 3 (DOM Integration)

```js
async function loadProducts() {
  const res = await fetch("/api/products");
  const products = await res.json();

  renderProducts(products);
}
```

---

# 4. POST Request (Create Data)

---

## Concept

Send data to server.

---

## Syntax

```js
fetch("/api/products", {
  method: "POST",
  headers: {
    "Content-Type": "application/json"
  },
  body: JSON.stringify(data)
});
```

---

## Example

```js
async function createProduct(product) {
  const res = await fetch("/api/products", {
    method: "POST",
    headers: {
      "Content-Type": "application/json"
    },
    body: JSON.stringify(product)
  });

  const result = await res.json();
  console.log(result);
}
```

---

# 5. PUT Request (Update Data)

---

## Concept

Update existing data.

---

## Example

```js
async function updateProduct(id, data) {
  await fetch(`/api/products/${id}`, {
    method: "PUT",
    headers: {
      "Content-Type": "application/json"
    },
    body: JSON.stringify(data)
  });
}
```

---

# 6. DELETE Request

---

## Concept

Remove data from server.

---

## Example

```js
async function deleteProduct(id) {
  await fetch(`/api/products/${id}`, {
    method: "DELETE"
  });
}
```

# 8. Handling Responses

```js
const res = await fetch(url);

  throw new Error("Request failed");
}
```

# 9. Handling JSON

```js
const data = await res.json();
```

# 10. Loading and Error Handling

---

## Example

```js
async function load() {
  try {
    showLoading();

    const res = await fetch("/api/products");

    if (!res.ok) throw new Error();

    const data = await res.json();

    render(data);

  } catch (err) {
    showError("Failed to load");
  } finally {
    hideLoading();
  }
}
```

# 12. Common Mistakes

❌ Forgetting `await res.json()`
❌ Not setting headers
❌ Not handling errors
❌ Using wrong HTTP method
❌ Not reloading data

---

# 13. Best Practices

* Always handle errors
* Always check `res.ok`
* Use async/await
* Separate API logic (service layer)
* Keep UI clean
