

# Lesson 5: User Input

## Phase 1: Programming Fundamentals

## Learning Objectives

By the end of this lesson you will be able to:

- Explain what user input is.
- Receive input in a browser using `prompt()`.
- Store input in variables.
- Understand that `prompt()` returns text.
- Convert text input to numbers.
- Validate simple user input.

---

# What Is User Input?

**User input** is information provided by a user to a program.

Examples:

- Typing a name
- Entering an age
- Searching for a product
- Filling out a form

Programs become interactive when they can receive input.

---

```

---

# Using `prompt()`

In a web browser:

```javascript
const name = prompt("What is your name?");
console.log(name);
```

If the user types:

```text
Alice
```

Output:

```text
Alice
```

`prompt()` displays a dialog box and returns what the user typed.

---

# Storing Input

```javascript
const city = prompt("Enter your city:");
console.log(city);
```

---

# Important: `prompt()` Returns a String

Even if the user enters:

```text
25
```

JavaScript receives:

```javascript
"25"
```

Notice the quotes—it's text.

---

# Converting to a Number

Use `Number()` when you need numeric input.

```javascript
const age = Number(prompt("Enter your age:"));

console.log(age);
```

Now mathematical operations work as expected.

```javascript
const a = Number(prompt("First number:"));
const b = Number(prompt("Second number:"));

console.log(a + b);
```

---

# Simple Validation

```javascript
const name = prompt("Enter your name:");

if (name === "") {
  console.log("Please enter a name.");
}
```

You'll learn `if` statements in detail soon.

---

# Common Beginner Mistakes

- Forgetting that `prompt()` returns text.
- Adding strings instead of numbers.
- Forgetting to store the result in a variable.

---

# Best Practices

- Use clear prompt messages.
- Convert numeric input with `Number()`.
- Validate important input before using it.

---

1. Ask the user for their name and print it.
2. Ask for their favorite color and print it.
3. Ask for two numbers, convert them with `Number()`, and print the sum.
4. Ask for their age and print: `"You entered: "` followed by the value.
5. Ask for a city and print a greeting using the entered city.

## Real-world Challenge

Build a simple "Greeting Program".

Requirements:

- Ask for the user's name.
- Ask for their age.
- Convert the age to a number.
- Print a personalized greeting.
- Print how old they are.

Example output:

```text
Welcome, Alice!
You are 25 years old.
```
